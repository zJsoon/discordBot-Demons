# Feature
- discord.js version -> V14.6.0
- Slash Commands
- .env -> Token protection system

## Commands

### Help

- /help

### Private

- /createverify
- /rules {example, anything that appears in .setDescription() can have its text changed to something else.}
- /announce (Soon...)

### Utils

- /avatar
- /info
- /ping
- /stats
- /userinfo
- /suggestion (Soon...)
- /lvl{XP} (Soon...)
- /lb{XP} (Soon...)
